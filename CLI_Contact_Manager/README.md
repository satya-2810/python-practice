# CLI Contact Manager

A simple Python CLI application to manage contacts.

## Features

* Add Contact
* View Contacts
* Search Contact by:

  * Name
  * Phone Number
* Delete Contact
* Input Validation

## Validations

* Name cannot be blank
* Phone number must:

  * contain only digits
  * be exactly 10 digits
* Email must contain `@`

## Run the Project

```bash
python contact_manager.py
```
