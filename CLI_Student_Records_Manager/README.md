# CLI Student Records Manager

A simple Command Line Interface (CLI) based Python application to manage student records using dictionaries and menu-driven operations.

## Features

* Add student records (Roll No, Name, Class, Marks)
* Display all student records
* Search student by Roll Number
* Update student marks
* Delete student records
* Input validation and error handling

## How to Run

1. Clone the repository
2. Run the Python file:

```bash
python student_records.py
```
